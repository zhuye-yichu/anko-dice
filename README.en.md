# 安科模拟骰子

#### Description
为了方便写安科文而写的一个工具，有以下功能：
（每类骰子都可以自定义每次骰子个数(num)、需要投多少次(fre)、每次骰子的值是否可以重复）

1、基本骰子：可自定义骰子大小(rd)，最小值为2；

2、权重骰子：可以自定义每一个数字或关键词的权重大小，可以自由增加或删除备选项；

3、范围区间骰子：可以拟定随机数字出现的区间范围，就是常见的设定min值，max值
