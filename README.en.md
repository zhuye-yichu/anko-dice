# 安科模拟骰子

#### Description
为了方便写安科文而写的一个工具，有以下功能：
1、基本骰子：可自定义骰子大小(rd)、每次骰子个数(num)、需要投多少次(fre)、每次骰子的值是否可以重复
2、权重骰子：可以自定义每一个数字或关键词的权重大小，根据权重和增加的数字或关键词数量来确定出现的概率
3、进阶骰子：可以自定义每一个数字或关键词每一批次内的最大重复次数、最小重复次数；
